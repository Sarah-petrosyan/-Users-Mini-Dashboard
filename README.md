## About the Project

This project is a responsive **user management dashboard** that allows adding, viewing, and managing users with their name, age, and status.

The dashboard includes features for **searching, filtering, and sorting users**, making it easy to organize and analyze user data in a clean interface.

The project focuses on practicing **JavaScript data handling and DOM manipulation**, demonstrating how dynamic lists, filters, and sorting logic can create an interactive dashboard experience.

This project was built to practice **interactive UI development, form validation, and real-time data updates**.

✅ Built with **HTML, CSS, JavaScript, and Bootstrap**  
✅ Add users with name, age, and status  
✅ Search users by name  
✅ Filter users by status (Student, Employed, Unemployed)  
✅ Sort users by name or age  
✅ Edelete user records
